== Description ==
The yahoo Buzz button easily allows your blog to be shared. 
Tested upto 2.9.1


#### Functionality
* New - yahoo buzz button. Integrates the ability to share your post with one click.


== Installation ==
Follow the steps below to install the plugin.

1. Upload the yahoo buzz directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in wp
3. Go to "yahoo Buzz" option to configure the button
